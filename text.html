关于2019级网络空间安全实验班学生选拔的通知

  url : https://jwc.xidian.edu.cn/info/1012/8922.htm

各有关单位：

2016年4月19日，习近平总书记在北京主持召开网络安全和信息化工作座谈会并发表重要讲话。关于人才培养习近平总书记强调，“网络空间的竞争，归根结底是人才竞争,互联网主要是年轻人的事业，要不拘一格降人才。要解放思想，慧眼识才，爱才惜才。培养网信人才，要下大功夫、下大本钱，请优秀的老师，编优秀的教材，招优秀的学生，建一流的网络与信息安全学院。互联网领域的人才，不少是怪才、奇才，他们往往不走一般套路，有很多奇思妙想。对待特殊人才要有特殊政策，不要求全责备，不要论资排辈，不要都用一把尺子衡量。”

为充分贯彻习总书记419讲话精神，落实中央网络安全和信息化领导小组办公室、国家发展和改革委员会、教育部、科学技术部、工业和信息化部、人力资源和社会保障部等中央六部委《关于加强网络安全学科建设和人才培养的意见》，服务国家信息安全的重大需求，培养网络空间安全拔尖人才，学校决定举办2019级网络空间安全实验班（以下简称实验班），由网络与信息安全学院管理。并计划在2019级本科生中选拔不超过40人，现将相关事宜通知如下：

一、报名资格

1.对网络空间安全有浓厚兴趣和特长的在读2019级理工类本科生。

2.国家招生政策及学校有关规定不能转专业的学生，不得申请。

3.具有高尚的爱国主义情操和集体主义精神，思想道德高尚，努力践行社会主义核心价值观，积极进取，品质优良，学风端正，表现良好，政治素养综合审查合格。

二、选拔办法

网络与信息安全学院成立选拔小组完成网络空间安全实验班学生的考核选拔工作，分为在线考核和面试两个环节。

1.申请报名学生在规定时间进行网上在线考核，根据在线考核成绩以及学生的主干课成绩（各占比50%）加权计算初试成绩并按从高到低依次排名，按计划选拔人数的150%比例确定进入面试环节的学生名单。

2.学院根据面试名单统一组织面试，重点考察申请学生的综合素质，在面试结束后公布拟录取名单。

3.成绩计算与录取办法

总成绩＝主干课程平均学分绩（占比40%）+在线考核成绩（占比40%）+面试成绩（占比20%）。

按照总成绩高低顺序排名，依次从高到低录取，录取人数不超过40人。

4.在线考核和面试内容

在线考核内容：信息安全与网络安全相关的基础学科知识，包括但不限于：计算机大类相关基础、网络与系统安全、应用安全、安全攻防对抗等，重点考核多种知识的综合运用与实践。

面试主要内容：自我介绍与现在(包括高考及原来学习工作、班级排名、学习基础、主要优势、申请转入实验班的目的、未来的志向等)、英语水平等。

5.对获得过相关竞赛奖励，确有专业特长的学生可酌情优先考虑。

三、报名及工作程序

1. 8月12日9:00-22:00，拟报考实验班的学生，在在线报名系统（http://ce.xidian.edu.cn/sub.htm）填写学生信息并报名；8月14日12:00，在网络与信息安全学院网站首页（http://ce.xidian.edu.cn/）公布报名学生信息和考试安排。

2. 8月16日9:00-9:30，学生熟悉在线考核系统；8月16日9:30-13:30，学生自主完成在线考核。

3. 8月19日，网络与信息安全学院根据选拔办法确定进入面试学生名单，报送本科生院审核备案后，在本科生院网站公布面试名单。

4. 8月22日，网络与信息安全学院组织面试考察。

5. 8月24日，网络与信息安全学院根据选拔办法确定拟录取学生名单，报本科生院审核。

6.学校对拟录取名单进行公示，公示期三天。学校在公示期内接受考生申诉，对申诉问题经调查属实的责成有关人员或专家小组复议。若考生对复议结果仍有异议，由本科生院组织有关专家进行复议。

7.经公示无异议后，本科生院正式发文公布实验班录取结果，办理相关转专业手续。

四、其他

1.实验班单独编班，创新培养方式，实施专用培养方案，学校及学院配备优秀任课教师，提供多方面条件支撑。

2.实验班学生培养、毕业及授位专业为“网络空间安全”专业。

3.从2020～2021学年第一学期开始，实验班学生的所有教学管理由网络与信息安全学院负责。

关于举办2020年校级物理实验竞赛的通知

  url : https://jwc.xidian.edu.cn/info/1012/8892.htm

各学院：

   为了配合2020年全国大学生物理实验竞赛活动，激发大学生对物理实验的兴趣，使学生广泛参与到物理实践中来，拟举办2020年西安电子科技大学物理实验竞赛，从中选拔出我校的参赛队参加全国大学生物理实验竞赛。具体安排如下：
一、 报名与参赛
1、参赛条件：在校本科生（2020年秋季学期在校）均可报名参赛，学生自由组队，每队不超过3人；每个学生只能参加一个参赛队，每个参赛队限选附件1中的一个题目参赛。
2、报名方式：
将参赛题目、姓名、班级、学号、专业、联系方式等信息（格式见附件3）邮件至 plc@mail.xidian.edu.cn 。
报名时间：即日起至2020年9月18日。
3、项目说明提交：每个参赛队必须在2020年9月18日下午五点前提交项目说明（论文组还需要提交论文，邮件提交即可）。格式要求见附件2，文件转换为pdf格式，并命名为“题目_负责人姓名_项目说明书.pdf”。说明书提交后仍然可以修改，答辩时自行打印4份提交给评委。
4、答辩时间：暂定2020年9月20日上午9：00。
5、答辩地点：暂定F楼215。
6、竞赛咨询：
竞赛官方 QQ 群：“西电物理实验竞赛 2020”，群号： “ 917417916 ”

二、 竞赛要求
制作类的作品要求：参赛作品应力求做到原理明确，装置简便，且易于操作，方法巧妙且手段新颖、有特色；现场操作规范，测量结果准确。应提交项目说明一份。
论文类的作品要求：论文限3~5页，按《物理实验》期刊格式撰写，答辩时请准备3~5页的PPT。讲解时尽量现场演示实验，如果不能现场演示的，PPT中应包含实验照片或录像。
三、 答辩要求
竞赛前，每队提前30分钟进入比赛场地调试和准备仪器。开始答辩后，首先自行介绍并演示10分钟，然后回答评委的提问。
四、 评奖
根据参赛总队数及评审结果，确定一、二、三等奖，并推荐有潜力的队伍参加全国大学生物理实验竞赛。

参加今年7月11日四六级考试的考生请看过来

  url : https://jwc.xidian.edu.cn/info/1012/8862.htm


各位考生：

本次考试起始时间：CET4：9：00～11：20

CET6：15：00～17：25

严禁将手机、平板电脑等通讯及可存储设备带入考场，考试结束前严禁离场！

1.本次英语四、六级考试听力部分南北校区均使用电台实施，请学生自行购买头戴式调频一体式耳机，电台工作频率是调频86.6MHZ。电台试音时间为早9点到晚18点，望各位学生及时试听并确保耳机工作正常（带够备份电池）。

2.所有考生从考前第14天开始，每日体温测量、记录并进行健康状况监测，体温测量记录以及出现身体异常情况的，要及时报告。出行时提前准备好口罩，做好个人防护，考生在进入考场前要佩戴口罩，进入考场后由考生自行决定是否佩戴。所有进入考试区域的考生必须接受体温测量，保持行进速度和间距。

所有考生体温低于37.3℃方可进入考点。第一次测量体温不合格的，可适当休息后使用其他设备或其他方式再次测量。仍不合格的以及有咳嗽等症状者，不得参加考试。

考生散场时要按监考员的指令有序离场，不得拥挤，保持人员间距。

3.CET考试在同一考场内使用多套试卷，试卷类型通过试题册背面的条形码粘贴条区分，所以提醒各位考生拿到试题册后务必将试题册背面上的条形码揭下贴到答题卡1上的指定位置！任何不贴条形码、不填写姓名、准考证号或条形码粘贴位置不对，均视为考试违规，评卷时将以“0”分计。请各位考生务必谨记！

敬告考生内容印在试题册正面，请考生在正式答题前仔细阅读所列内容，并在考试过程中自觉遵守考试纪律和考场规则，不得携带手机等通信工具进入考场，不得中途离开考场。对于利用手机、无线耳机等通讯工具作弊的考生，将按教育部有关规定给予开除学籍处分。

再次重申不得将答题卡、试题册带出考场，考试结束前严禁离开考场。

4.在试题册背面增加了考生姓名及准考证号填写等事项；作文题目也在试题册背面。

 

祝各位考生取得优异的成绩！

关于公布2020年西安电子科技大学第六届英语写作、阅读大赛成绩的通知

  url : https://jwc.xidian.edu.cn/info/1012/8832.htm

各学院：
  我校于2020年6月20日--6月21日成功举办了2020年西安电子科技大学第六届英语“写作之星”、第六届英语阅读大赛决赛。这两个赛事的决赛于线上进行，共计180余名学生参加了决赛。最终，经过机器评阅和专家评委教师评阅、审核和复议，各赛事分别评出特等奖3名、一等奖5名、二等奖8名、三等奖14名。所有获奖证书和奖品将于下学期发（请关注后续通知）。
  现将获奖名单予以公布。

附件2：西安电子科技大学第六届英语阅读大赛获奖名单

关于2020年（2016级）第二批本科生毕业设计（论文）盲审及答辩工作安排的通知

  url : https://jwc.xidian.edu.cn/info/1012/8812.htm

 

关于2020年（2016级）第二批本科生

毕业设计（论文）盲审及答辩工作安排的通知

 

各学院：

根据我校《西安电子科技大学关于2016级本科生毕业设计（论文）工作安排的通知》(本字〔2019〕177号)及《关于做好2019～2020学年第二学期毕业年级学生教学工作安排的通知》(本字〔2020〕24号)的安排，现将7月份盲审和毕业答辩工作安排通知如下。

一、盲审时间安排及说明

（一）7月15日（周三）中午12:00前，未参加第一批盲审的学生（87人）通过系统提交盲审论文全部材料。

（二）7月15日（周三）下午至17日（周五），学校将统一组织开展盲审检查工作。

（三）7月20日（周一），根据盲审检查结果,形成《关于公布2020年（2016级）第二批毕业设计（论文）盲审结果及相关事项的通知》（7月），并同步发布在教务处主页上。

（四）盲审说明及盲审材料详单请见《关于2020年（2016级）本科生毕业设计（论文）盲审工作安排的通知》(本字〔2020〕32号)文件（附件1）。

二、毕业答辩时间安排

（一）各学院于7月21日（周二）中午12:00前报送第二批答辩安排，并于7月26日（周日）之前完成毕业设计（论文）答辩工作。答辩结束后，于7月31日（周五）前完成毕设答辩成绩和归档材料上传及审核工作。

（二）毕业答辩组织要求及归档材料上传要求等请见《关于2016级毕业设计（论文）答辩工作注意事项的通知》（附件2）及《西安电子科技大学本科生毕业设计（论文）工作条例（修订）》(本字〔2016〕125号)（附件3）。
